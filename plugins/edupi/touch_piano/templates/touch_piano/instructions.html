{% trans "Wiring Explanation" %}

{% trans "Important:" %} {% trans "The 1MΩ resistors are optional but recommended. They help stabilize the input signals and reduce false triggers from electrical noise." %}

{% trans "Each GPIO pin acts as an input with an internal pull-up resistor. When you touch the conductive material connected to a GPIO pin, your body creates a path to ground, pulling the pin low. This change is detected by the software and triggers the corresponding piano note." %}

{% trans "Troubleshooting" %}

  • {% trans "If keys are too sensitive or triggering randomly, try increasing the sensitivity setting in the configuration." %}
  • {% trans "If keys don't respond, ensure all connections are secure and the conductive material is clean." %}
  • {% trans "For better touch detection, make sure you have a good ground connection (touch the ground pin or hold a grounded metal object)." %}
  • {% trans "If using fruits, make sure they are fresh and not dried out." %}
